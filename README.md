### Как разблокировать сайты

1. Откройте инструмент Прокси и VPN
2. Нажмите на вкладку **"Разблокировка"**
3. Выберите режим **"Разблокировка сайтов"**

5. Нажмите **"Подключиться"**

### Как использовать обход DPI

1. Нажмите на вкладку **"Обход DPI"**
2. Выберите метод: **Goodbye DPI** или **DPITunnel**
3. Включите **"Fragment Bypass"** для фрагментации TCP
4. Включите **"Host Spoofing"** для подмены домена
5. Нажмите **"Запустить обход"**

### Как использовать VPN для игр

1. Нажмите на вкладку **"Игровой режим"**
2. Выберите **"Разблокировка игр"**
3. Выберите игровую платформу (Steam, Epic и т.д.)
4. Выберите сервер с наименьшим пингом

### Как использовать прокси для браузера

1. Нажмите на вкладку **"Прокси"**
2. Выберите тип прокси (HTTP, HTTPS, SOCKS5)
3. Выберите местоположение прокси
4. Нажмите **"Получить прокси"**
5. Скопируйте IP и порт
6. Настройте в браузере прокси

### Как автоматически менять IP

4. Нажмите **"Запустить ротацию"**
5. IP будет меняться автоматически
